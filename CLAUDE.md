## Tools

### Project-wide

- `make build`: Build source distribution and wheel
- `make check`: Run type checks and linters
- `make docs`: Build documentation
- `make fix`: Run linters and fix errors
- `make format`: Run formatters
- `make lint`: Run linters
- `make pre-commit`: Run pre-commit hooks on changed files
- `make pre-commit-all`: Run pre-commit hooks on all files
- `make types`: Run type checks
- `make test`: Run all tests (includes coverage)
- `make test-unit`: Run unit tests
- `make test-e2e`: Run end-to-end tests
- `make test-integration`: Run integration tests
- `make test-benchmark`: Run benchmark tests
- `make test-performance`: Run performance tests
